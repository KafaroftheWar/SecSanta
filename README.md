# Telegram Бот "Тайный Санта"

Бот для организации игры "Тайный Санта" - случайного распределения номеров участников для обмена подарками.

## Описание

Этот Telegram-бот позволяет участникам получить случайное число от 1 до 26, которое соответствует номеру в списке участников. Каждый пользователь может получить только одно число, которое сохраняется до сброса базы данных администратором.

## Команды бота

### `/start`
Запускает бота и выводит приветственное сообщение с инструкциями.

**Использование:**
```
/start
```

**Что делает:**
- Приветствует пользователя
- Объясняет, как получить число
- Напоминает важные правила участия

### `/getnumber`
Получает случайное число из доступных (от 1 до 26).

**Использование:**
```
/getnumber
```

**Что делает:**
- Выдает случайное число, которое еще не было присвоено
- Сохраняет число за пользователем в базе данных
- Каждый пользователь может получить только одно число

**Ограничения:**
- Если пользователь уже получил число, при повторном использовании команды будет выдана ошибка: "Нельзя получить больше 1 числа"
- Если все числа уже были использованы, будет сообщение: "Все числа уже были использованы!"

**Важно:**
- Полученное число сохраняется до сброса базы данных администратором
- Нельзя получить новое число, пока база данных не будет сброшена
- При повторном использовании команды будет выдана ошибка

### `/reset_adminapproved` (только для администраторов)
Сбрасывает базу данных и восстанавливает все числа (от 1 до 26) для нового розыгрыша.

**Использование:**
```
/reset_adminapproved
```

**Доступ:**
- Команда доступна только администраторам с определенными Telegram ID
- Обычные пользователи получат сообщение: "Отказано в доступе"

**Что делает:**
- Удаляет все записи из базы данных
- Восстанавливает все числа от 1 до 26
- Позволяет участникам получить новые числа

## Правила участия

1. **Конфиденциальность:** Никому не сообщайте выпавший номер, это может разрушить всё мероприятие.

2. **Стоимость подарка:** Подарок должен стоить в районе 300 рублей (или в районе оговоренной суммы).

3. **Одно число на пользователя:** Каждый участник может получить только одно число. Повторное использование команды `/getnumber` приведет к ошибке.

## Технические детали

- **База данных:** SQLite (`numbers.db`)
- **Диапазон чисел:** 1-26
- **Логирование:** Все действия пользователей логируются с временными метками
- **Безопасность:** Административные команды защищены проверкой Telegram ID
